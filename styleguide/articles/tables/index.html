Допоміжні класи для кольорів (ставляться на тег th або td):
  • highlighted-blue - синій колір для тексту
  • highlighted-red - червоний колір тексту
  • background-white - білий колір для комірки (для td)
  • blue-gradient - синій градієнт
  • blue-gradient-1 - синій світлий градієнт (може йти в парі з blue-gradient-2, для наступних комірок)
  • blue-gradient-2 - синій темніший градієнт (може йти в парі з blue-gradient-1, для наступних комірок)
  • yellow-gradient - жовтий градієнт
  • background-blue-1 - синій колір для комірки (світліший)
  • background-blue-2 - синій колір для комірки (темніший)
